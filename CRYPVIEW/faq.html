🚀 Démarrage
CrypView est-il gratuit ? Faut-il un compte ?

Oui, CrypView est entièrement gratuit et ne nécessite aucun compte, aucune inscription et aucune carte bancaire. Ouvrez l'URL et commencez immédiatement.

L'application n'a aucun modèle freemium — toutes les fonctionnalités sont accessibles sans condition.

Quels navigateurs sont supportés ?

CrypView fonctionne sur tous les navigateurs modernes :

  • Chrome 90+ (recommandé pour l'installation PWA)
  • Firefox 88+
  • Edge 90+
  • Safari 15+ (iOS 15+ pour la PWA)

Internet Explorer n'est pas supporté.

Comment changer de paire de trading ?

Cliquez sur le champ de recherche dans le header (ou appuyez sur Ctrl+K) et tapez le nom du token (ex: ETH, SOL). Sélectionnez la paire dans la liste déroulante ou validez avec Entrée.

En mode multi-graphiques, chaque panneau possède son propre champ de recherche.

Comment basculer entre thème sombre et clair ?

Faites un clic droit sur le graphique → Outils → Paramètres → choisissez le thème. La préférence est sauvegardée localement et appliquée instantanément sans rechargement.

📊 Graphiques & Multi-Charts
Quels timeframes sont disponibles ?

CrypView propose 15 timeframes : 1s, 1m, 3m, 5m, 15m, 30m, 1h, 2h, 4h, 6h, 12h, 1j, 3j, 1sem, 1mois.

Appuyez sur T pour cycler rapidement entre les timeframes.

Comment utiliser le mode Multi-Charts ?

Clic droit → Multi-Charts → choisissez la disposition : 2 côte à côte, 4 en grille, 9 en 3×3, 2 ou 3 verticaux, ou les layouts asymétriques 1+2 et 1+3.

Chaque panneau est indépendant : symbole, timeframe et indicateurs propres. La synchronisation du crosshair et du zoom est activable via la barre de sync en haut.

Comment utiliser les Drawing Tools ?

Clic droit → Drawing Tools → choisissez l'outil. Les tracés disponibles sont : Trendline, Fibonacci, Zone horizontale, Rectangle, Pitchfork, Flèche, Texte, Mesure, Canal parallèle.

Les ancres sont déplaçables par glisser-déposer (v2.9+). Faites un clic droit sur un tracé pour obtenir des options (alertes de croisement, suppression). Appuyez sur Ctrl+Z pour annuler le dernier tracé.

Les tracés sont-ils sauvegardés ?

Oui, tous les tracés sont sauvegardés dans le localStorage de votre navigateur. Ils survivent aux rechargements de page et sont associés à chaque symbole (synchronisation MTF automatique).

Attention : vider le cache du navigateur supprimera les tracés.

📈 Indicateurs
Quels indicateurs sont disponibles ?

CrypView propose 19+ indicateurs organisés en 4 catégories :

  • Tendance : MA 20/50/200, VWAP, HMA, Ichimoku, SuperTrend, Keltner, EMA Multi, DEMA, Parabolic SAR, Donchian, Régression Linéaire, Pivot Points
  • Momentum : RSI, MACD, Stochastique, CCI, ADX, Williams %R, Momentum, TRIX, Squeeze Momentum, Elder Ray
  • Volatilité : Bollinger, ATR
  • Volume : Volume Profile, Footprint, Orderflow Delta/CVD, MFI, OBV, CMF
Comment ajouter / supprimer un indicateur ?

Clic droit → Indicateurs → Choisir les indicateurs, ou appuyez sur I. La palette de commandes (Ctrl+K) permet aussi de les activer/désactiver rapidement.

Raccourcis directs : Shift+R (RSI), Shift+M (MACD), Shift+B (Bollinger), Shift+V (VWAP), Shift+F (Footprint).

Qu'est-ce que le Footprint Chart ?

Le Footprint affiche la pression acheteur (Ask) et vendeur (Bid) par niveau de prix à l'intérieur de chaque bougie. Les déséquilibres (ratio ≥ 3×) sont signalés par une bordure dorée.

Les données sont amorcées via REST (aggTrades historiques) puis mises à jour en temps réel par WebSocket. Le niveau de zoom minimum pour afficher les valeurs est de ~28 pixels par bougie.

Comment utiliser les Profils & Presets ?

Les profils permettent de sauvegarder une combinaison d'indicateurs + timeframe. Clic droit → Indicateurs → Profils & Presets. 6 presets intégrés (Scalping, Swing, Volume, Orderflow, Momentum, Volatilité) et jusqu'à 20 profils personnalisés.

🔔 Alertes de prix
Comment créer une alerte ?

Faites un clic droit sur le graphique au niveau souhaité → Alertes → Nouvelle alerte. L'alerte est automatiquement pré-remplie avec le prix du curseur.

Il est aussi possible de cliquer droit sur une trendline tracée pour créer une alerte de croisement (↑ ou ↓).

Les alertes fonctionnent-elles quand l'onglet est fermé ?

Non. CrypView est une application web locale sans backend — les alertes nécessitent que l'onglet soit ouvert (même en arrière-plan). Installez CrypView en PWA pour que l'onglet reste actif facilement.

Quand l'onglet est en arrière-plan, les WebSockets restent actifs et les alertes continuent de fonctionner.

Quels types de conditions sont supportés ?

11 types de conditions combinables en logique AND/OR :

  • Prix absolu (au-dessus/en-dessous d'un seuil)
  • Variation % depuis l'ouverture 24h
  • Volume spike (× multiple de la moyenne)
  • RSI (au-dessus ou en-dessous d'un seuil)
  • Croisement MACD (haussier ou baissier)
  • Breakout N-bougies (résistance ou support)
  • Croisement de trendline tracée (↑ ou ↓)
📲 Installation PWA
Comment installer CrypView sur mon appareil ?

Chrome / Edge (desktop) : cliquez sur l'icône ⊞ dans la barre d'adresse → Installer.

Safari iOS : bouton Partager → Ajouter à l'écran d'accueil.

Firefox : Menu → Installer l'application.

La PWA fonctionne-t-elle sans connexion internet ?

Les assets statiques (HTML, CSS, JS, polices) sont mis en cache et disponibles hors ligne. Cependant, les données de marché (Binance WebSocket) nécessitent une connexion internet active.

📡 Données & Confidentialité
D'où viennent les données de marché ?

Toutes les données proviennent de l'API publique Binance (REST + WebSocket). CrypView n'agrège, ne stocke ni ne redistribue aucune donnée — chaque requête part directement de votre navigateur vers Binance.

Les données DEX proviennent de GeckoTerminal (API publique gratuite).

Mes données sont-elles envoyées à un serveur ?

Non. CrypView est une application 100% client-side (aucun backend). Vos alertes, tracés, paramètres et historique Paper Trading sont stockés localement dans votre navigateur (localStorage). Aucune donnée ne transite par nos serveurs.

Combien de paires sont disponibles ?

Toutes les paires USDT actives sur Binance Spot (500+). La liste est chargée au démarrage depuis GET /api/v3/exchangeInfo et mise en cache pour la session.
