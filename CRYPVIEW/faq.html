🗺 Général
Qu'est-ce que CrypView ?+
CrypView est une application web gratuite, 100% côté client…
CrypView est-il gratuit ?+
Oui, entièrement.
Dois-je créer un compte ?+
Non. CrypView ne requiert aucune inscription…
Puis-je installer CrypView sur mon appareil ?+
Oui. CrypView est une Progressive Web App (PWA)…
CrypView fonctionne-t-il sur mobile ?+
Oui. CrypView est entièrement responsive…
Quelles langues sont disponibles ?+
CrypView supporte 4 langues…
📊 Données & Graphiques
D'où proviennent les données de marché ?+
Les données de prix proviennent directement de l'API publique Binance…
Les données sont-elles en temps réel ?+
Oui. Les données de chandeliers arrivent via WebSocket…
Quelles paires de trading sont disponibles ?+
Toutes les paires USDT actives listées sur Binance Spot…
Quels timeframes sont supportés ?+
15 timeframes : 1s · 1m · 3m · 5m…
Quels types de graphiques sont disponibles ?+
Chandeliers japonais, Heikin-Ashi, Ligne et Barres…
Comment accéder aux graphiques DEX ?+
Clic droit → DEX, ou cliquez sur le bouton 🔗 DEX…
Comment ouvrir plusieurs graphiques en même temps ?+
Clic droit → Multi-Charts, ou utilisez les liens de layout…
📈 Indicateurs
Comment ajouter un indicateur ?+
Clic droit → Indicateurs → Choisir des indicateurs…
Combien d'indicateurs sont disponibles ?+
Plus de 19 : MA, EMA, DEMA, HMA, VWAP, Ichimoku…
Certains indicateurs sont lents à charger — est-ce normal ?+
Oui. Les indicateurs lourds sont calculés dans un Web Worker…
Qu'est-ce que les profils d'indicateurs ?+
Les profils sauvegardent une combinaison d'indicateurs actifs…
🔔 Alertes
Comment créer une alerte de prix ?+
Clic droit sur le graphique → Alertes → Nouvelle alerte…
Comment suis-je notifié quand une alerte se déclenche ?+
De deux façons simultanément…
Quelles conditions puis-je définir sur les alertes ?+
Prix au-dessus/en dessous, hausse/baisse ≥ %…
Les alertes fonctionnent-elles avec l'onglet en arrière-plan ?+
Oui, tant que l'onglet reste ouvert…
Les alertes sont-elles sauvegardées si je ferme le navigateur ?+
Oui. Les alertes sont sauvegardées dans le localStorage…
⚗️ Paper Trading & Backtesting
Qu'est-ce que le Paper Trading ?+
Le Paper Trading est un moteur de simulation sans risque…
Les résultats du Paper Trading peuvent-ils prédire le trading réel ?+
Non. Le Paper Trading ne modélise pas le slippage…
Qu'est-ce que le Backtesting ?+
Le Backtesting exécute une stratégie sur des bougies OHLCV historiques…
Les résultats de backtest sont-ils fiables ?+
Les backtests sont indicatifs seulement…
🔒 Technique & Confidentialité
CrypView collecte-t-il mes données personnelles ?+
Non. CrypView est 100% côté client…
Qu'est-ce qui est stocké dans mon navigateur ?+
Préférence de thème, langue, alertes de prix…
Comment supprimer mes données ?+
Ouvrez les DevTools → Application → Local Storage…
CrypView fonctionne-t-il hors ligne ?+
Partiellement. Les ressources statiques sont mises en cache…
Ai-je besoin d'une clé API Binance ?+
Non. CrypView utilise uniquement les endpoints publics de l'API Binance…
CrypView donne-t-il des conseils en investissement ?+
Non. CrypView est uniquement un outil d'analyse…
