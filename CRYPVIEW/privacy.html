Politique de confidentialité

Dernière mise à jour : Juin 2025  ·  CrypView — Beta Capital Enterprise

🔒 CrypView est une application 100% client-side. Aucune donnée personnelle n'est collectée, transmise ou stockée sur nos serveurs — car CrypView n'en possède pas. Tout fonctionne localement dans votre navigateur.

🏢 Responsable du traitement

CrypView est édité par Beta Capital Enterprise. Pour toute question relative à la présente politique, vous pouvez nous contacter à l'adresse suivante : betacapital.discord@gmail.com

CrypView est hébergé sur GitHub Pages (GitHub, Inc., 88 Colin P. Kelly Jr. Street, San Francisco, CA 94107, USA). L'accès au service implique l'application de la politique de confidentialité de GitHub concernant les données de trafic serveur.

📋 Données collectées

CrypView ne collecte aucune donnée personnelle identifiable. L'application ne dispose d'aucun formulaire d'inscription, d'aucun système de compte utilisateur, et n'intègre aucun outil d'analyse comportementale (Google Analytics, Hotjar, Mixpanel, etc.).

🌐 Données de trafic (hébergeur)

Comme tout site web, l'accès à CrypView génère automatiquement des journaux d'accès côté serveur, gérés par GitHub Pages. Ces journaux peuvent inclure votre adresse IP, le navigateur utilisé, la page consultée et l'horodatage. Ces données sont traitées par GitHub conformément à leur politique de confidentialité et ne sont pas accessibles à Beta Capital Enterprise.

CrypView ne possède aucun serveur applicatif propre et ne reçoit aucune donnée de trafic.

🔌 Services tiers intégrés

CrypView sollicite les services externes suivants. Votre navigateur effectue directement ces requêtes — CrypView n'agit pas en intermédiaire :

  • Binance API (api.binance.com) — données de marché OHLCV, WebSocket temps réel. Politique Binance.
  • GeckoTerminal API (api.geckoterminal.com) — données DEX si la fonctionnalité est activée. Politique CoinGecko.
  • Google Fonts (fonts.googleapis.com) — chargement des polices DM Mono et Syne. Politique Google.
  • GitHub Pages — hébergement statique. Politique GitHub.

CrypView ne partage aucune donnée avec des régies publicitaires, des courtiers de données ou des partenaires commerciaux.

💾 Stockage local (localStorage)

L'ensemble des données de fonctionnement de CrypView est stocké dans le localStorage de votre navigateur. Ce mécanisme est strictement local à votre appareil — aucune donnée ne transite par un réseau ou un serveur tiers.

Vous pouvez supprimer intégralement ces données à tout moment :

  • Via les DevTools du navigateur → Application → Local Storage → effacer les clés crypview_*
  • En vidant le cache et les données de navigation (Paramètres → Confidentialité)
  • En utilisant le mode navigation privée (les données ne sont pas persistées)

⚖️ Vos droits (RGPD)

En l'absence de collecte de données personnelles identifiables, les droits d'accès, de rectification et d'effacement prévus par le RGPD s'appliquent de facto : toutes vos données sont accessibles et supprimables directement depuis votre navigateur, sans intermédiaire.

Si vous estimez que vos droits ne sont pas respectés, vous pouvez introduire une réclamation auprès de la CNIL (Commission Nationale de l'Informatique et des Libertés) : www.cnil.fr.
